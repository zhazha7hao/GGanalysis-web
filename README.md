# 抽卡概率计算器 (Gacha Probability Calculator)

这是一个静态网页应用，用于计算原神、星穹铁道和绝区零的抽卡概率。算法彻底移植自 `GGanalysis` Python 库，确保**高精度**和**严格符合游戏机制**。

## 核心功能

1.  **高精度算法**:
    *   **原神**: 完整实现了 5.0 版本的“捕获明光”机制 (Capturing Radiance DP)，而非简单的概率近似。
    *   **星穹铁道**: 严格采用 56.25% / 78.125% 的综合概率模型。
    *   **绝区零**: 包含精确的音擎池参数 (65 软保底)。

2.  **可视化增强**:
    *   **双轴图表**: 左轴显示“单次概率 (PDF)”，右轴显示“累计概率 (CDF)”，您可以直观看到“多少抽能达到 90% 成功率”。

3.  **极简交互**:
    *   **自动计算**: 修改任意参数，结果实时更新。
    *   **统一界面**: 同时支持同游戏下的角色与武器混合计算。

## 使用说明

1.  **打开**: 双击 `index.html`。
2.  **选择游戏**: 点击上方标签切换游戏。
3.  **输入**: 
    *   将不想抽的项目填 **0**。
    *   填入垫抽数和大保底状态。
4.  **查看结果**: 
    *   上方数值卡片显示期望投入。
    *   下方图表的绿色虚线代表累计成功率 (CDF)。

## 技术说明

本项目是一个纯前端应用，所有计算都在浏览器中毫秒级完成。
代码位于 `js/logic/` 目录下，其中 `model.js` 包含了复杂的动态规划 (DP) 逻辑以处理原神的特殊机制。
